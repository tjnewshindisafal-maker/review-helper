🌐 Step 1 — Create Free JSONBin Database

1
Go to jsonbin.io → Sign up free
2
Click New Bin → Paste this as content:
{"users":[]}
3
Copy your Bin ID (from URL bar) and Master Key (from Account > API Keys)

⚙️ Step 2 — Update Both Files

1
Open advizr-saas.html and admin.html in Notepad
2
Find this line in both files:
var DB_MODE = 'local';
3
Change to:
var DB_MODE = 'jsonbin';
4
Also update these 2 lines with your values:
var JSONBIN_KEY = 'YOUR_KEY_HERE';
var JSONBIN_BIN = 'YOUR_BIN_ID';

🚀 Step 3 — Deploy on Hostinger

1
Login to Hostinger hPanel → File Manager
2
Go to public_html folder
3
Upload advizr-saas.html → Rename to index.html
4
Upload admin.html → Keep as admin.html
5
Access: yoursite.com (users) & yoursite.com/admin.html (you)

💳 Step 4 — Setup Razorpay Payments

1
Go to razorpay.com → Create free account
2
Create a Payment Link for ₹499/month
3
Share Razorpay link on WhatsApp when user requests upgrade
4
After payment, go to Admin Panel → Payments → Mark as Paid → Plan auto-upgrades to Pro
